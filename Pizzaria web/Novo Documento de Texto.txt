Frontend: Design responsivo com navegação entre as páginas Início, Menu e Sobre. Carregamento dinâmico de menus via API, carrinho de compras interativo com persistência localStorage e confirmações modais estilizadas substituindo os prompts do navegador. Backend: Servidor Express.js que serve arquivos estáticos e endpoints de API RESTful. Banco de dados SQLite com tabelas de categorias e produtos, inicializado com dados de pizza de exemplo. Experiência de compra: os usuários podem navegar pelas pizzas por categoria, adicionar itens ao carrinho, ajustar quantidades e concluir pedidos por meio de um processo de checkout simulado com modais de confirmação. Pilha técnica: HTML5, CSS3, JavaScript ES6+, Node.js, Express.js, SQLite3. Apresenta design responsivo, animações suaves e padrões modernos de UI/UX. O projeto serve como um exercício prático em desenvolvimento web, mostrando integração de banco de dados, design de API, comunicação frontend-backend e implementação de interface de usuário para um cenário de comércio eletrônico.

Aplicativo Pizzaria O Aplicativo Pizzaria é uma solução de e-commerce em React Native com TypeScript e Expo Go, compatível com Android e iOS, oferecendo uma experiência moderna para pedidos de pizza. Funcionalidades Autenticação Cadastro com validação (Nome, Login, Senha, Email, Número, Endereço) Login/Logout com AsyncStorage Validação de duplicatas (login/email único) Interface adaptativa com saudação personalizada Catálogo Categorias: Pizzas, Bebidas, Combos, Doces Produtos em destaque e filtros dinâmicos Layout responsivo (2 produtos/linha, cards uniformes) Interface Design system: Azul, gradientes, sombras Tipografia, animações suaves Carrinho Adição/remoção de produtos, controle de quantidade Cálculo automático do total, persistência local Interface modal para melhor UX Tecnologias React Native, TypeScript, Expo Go AsyncStorage, Context API, Expo Linear Gradient MySQL (tabelas: users, products, orders, order_items) Produtos Pizzas: Frango com Catupiry, Quatro Queijos, Camarão, Chocolate com Morango Bebidas: Coca Cola, Guaraná, Água Combos: Família, Vegetariano, Kids, Amigos, Individual, Duplo Características Técnicas Design: Cores azul escuro, gradientes, bordas arredondadas (16-25px) Responsividade: Layout adaptativo, scroll horizontal Fluxo do Usuário Entrada → Catálogo → Filtros → Detalhes → Carrinho →Confirmação Valor.

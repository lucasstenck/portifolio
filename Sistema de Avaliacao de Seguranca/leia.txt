Projeto: Plataforma web de avaliação NIST CSF com painel administrativo.
Funcionalidades principais
Avaliações (assessments): criação, edição, visualização detalhada por funções/subcategorias; cálculo de pontuações e maturidade.
Clientes: cadastro, gerenciamento e geração de relatórios (Word com PHPWord usando nome da empresa configurado).
Relatórios (reports): estatísticas gerais, tabelas e gráficos (linha, radar, pizza, barras) com exportação CSV/PDF.
Dashboard: visão geral, temas e configurações visuais.
LGPD: botão “Política de Privacidade” em clients.php com modal editor; conteúdo sincronizado no banco (site_config) e no arquivo assets/docs/termos_lgpd.php; busca no modal; responsivo.
Configurações (settings): personalização de cores (inclui cor do nome da empresa no rodapé), fontes e dados de contato.
Usuários: gestão e permissões (admin, gerente de downloads etc.).
Download: central de downloads.
Frontend público (index/resultado): validação de LGPD (mensagem de erro), aplicação dinâmica de cores do rodapé.
Tecnologias e ferramentas
Backend: PHP (XAMPP), PDO/MySQL, sessões, controle de acesso.
Banco de dados: MySQL (db nist_assessment), tabela site_config para chaves/valores de configuração.
Bibliotecas:
Chart.js (gráficos na web e renderização no PDF via Browsershot).
Spatie Browsershot (Chrome/Node) para gerar PDF de reports.
PHPWord para DOCX de clientes.
Frontend: HTML5, CSS (estilos utilitários/inline), JavaScript vanilla (modais, validações, busca no editor).

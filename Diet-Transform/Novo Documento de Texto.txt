Este projeto implementa um funil de vendas completo para produtos dietéticos, com três páginas principais para capturar leads e converter visitantes. A landing page (index.html) apresenta um design moderno e responsivo, com seções de destaque, benefícios, depoimentos e uma garantia. Após 1 minuto, um modal de captura aparece automaticamente para coletar dados. A página de registro/login (registration.html) possui um sistema de autenticação com validação de senha em tempo real, botões de visibilidade e armazenamento seguro. Após o login, o usuário é redirecionado para a área de membros. A área de membros (members.html) oferece um painel de progresso, cursos em vídeo, downloads, um sistema de conquistas e rastreamento de atividades. O projeto utiliza HTML5, CSS3 e JavaScript ES6+, com design responsivo, Font Awesome e Google Fonts (Inter). O backend é MySQL com dois bancos de dados: um para usuários e outro para leads. O sistema de análise rastreia o comportamento do usuário e as conversões, com notificações em tempo real. O LocalStorage armazena os dados como backup. Cores principais: azul (#667eea), roxo (#764ba2), verde (#4ade80). A segurança inclui validação de formulários e proteção de rotas. Ideal para marketing digital, cursos online e produtos de informação.

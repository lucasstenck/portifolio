Uma landing page moderna e responsiva, projetada para capturar leads por meio de um sistema modal inteligente. Desenvolvido com HTML5, CSS3 e JavaScript, com backend PHP e integração com banco de dados MySQL. Principais características: Design responsivo: layout totalmente responsivo que se adapta perfeitamente a desktops, tablets e dispositivos móveis usando CSS Grid, Flexbox e tipografia fluida com funções clamp(). Modal de Captura Inteligente de Leads: Aparece automaticamente após 60 segundos de interação do usuário Captura nome, e-mail e número de telefone com validação em tempo real Mascaramento de número de telefone para melhor UX Evita envios duplicados usando localStorage Combina a estética do design do site com fundos gradientes e animações Integração de backend: Script PHP para processamento de formulários com conexão de banco de dados PDO Banco de dados MySQL com indexação e restrições adequadas Sanitização e validação de entradas Prevenção de e-mails duplicados Manipulação de resposta JSON UI/UX moderna: Animações e transições suaves Efeitos interativos de foco Estados de carregamento e feedback do usuário Notificações de sucesso/erro Recursos de acessibilidade (navegação por teclado, suporte para movimento reduzido) Pilha técnica: Frontend: HTML5, CSS3 (Grid/Flexbox), JavaScript básico ES6+ Backend: PHP 7.4+, MySQL 5.7+ Recursos: Intersection Observer, localStorage, API de busca Design: Google Fonts, animações CSS, fundos gradientes Otimizações de desempenho: Carregamento lento para imagens

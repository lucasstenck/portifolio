Um conversor de moedas desenvolvido com Node.js e Express, oferecendo conversão em tempo real entre 150 moedas diferentes. O aplicativo combina uma API com uma interface web. Características: Conversão entre 150 moedas globais Atualizações automáticas a cada 5 minutos via API externa Interface moderna com design de morfismo de vidro e gradientes Pesquisa inteligente e filtros para fácil seleção Seção de moedas populares para acesso rápido Indicador de status online/offline em tempo real Design totalmente responsivo para dispositivos móveis e desktop Tecnologias utilizadas: Backend: Node.js, Express, Axios, Node-Cron Frontend: HTML5, CSS3, JavaScript puro Interface: Design moderno com gradientes escuros, efeitos de transparência, animações fluidas e tipografia elegante. Interface intuitiva com cartões interativos, botões com efeitos de foco e indicadores visuais de status. Capacidade de resposta: Totalmente adaptável a dispositivos móveis, tablets e desktops, mantendo uma experiência visual consistente em todas as telas. Ideal para desenvolvedores, viajantes, investidores e qualquer pessoa que precise de conversões de moeda precisas e em tempo real.

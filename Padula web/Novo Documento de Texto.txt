Padula Accounting - Sistema de Captura de Leads com Modal WhatsApp Este projeto implementa um sistema de captura de leads para o site da Padula Accounting, integrando um modal que aparece quando o usuário clica no botão do WhatsApp. O sistema combina um front-end moderno com um back-end robusto para uma experiência de usuário fluida. Principais características: Modal Inteligente: Quando o visitante clica no botão do WhatsApp, um modal solicita seu nome, e-mail e número de telefone. É responsivo, segue a identidade visual da empresa e oferece animações fluidas com feedback em tempo real. Back-end Node.js: O Express Server com API REST processa e valida os dados dos leads. Um banco de dados SQLite garante armazenamento persistente e a validação dupla (front-end e back-end) protege os dados. Integração com o WhatsApp: Após o envio, o sistema redireciona o usuário para o WhatsApp com uma mensagem pré-formatada contendo seus dados para facilitar o primeiro contato. Validação robusta: a validação em várias camadas garante o tratamento adequado dos dados, com tratamento de erros e feedback em tempo real. Interface moderna: design responsivo com animações CSS3, estados visuais claros (carregamento, sucesso, erro). Tecnologias utilizadas: Front-end: HTML5, CSS3, JavaScript Back-end: Node.js, Express.js Banco de dados: SQLite Benefícios: Captura automática de leads Reduz a perda de contato Fácil integração com o WhatsApp Interface profissional e moderna Fácil manutenção e escalabilidade

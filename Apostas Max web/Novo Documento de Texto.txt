ApostasMax - Portal Completo de Loterias ApostasMax é uma plataforma web desenvolvida em Python com Flask que oferece ferramentas especializadas para apostadores de loteria brasileiros. O projeto é estruturado como um portal central com múltiplos microsserviços, cada um dedicado a funcionalidades específicas. Arquitetura principal: O backend principal utiliza Flask com PyMySQL para se conectar a um banco de dados MySQL, gerenciando a autenticação do usuário, sessões persistentes e roteamento para diferentes seções. O sistema implementa um despachador de middleware que direciona solicitações para subaplicativos específicos com base na URL. Principais características: Geradores Inteligentes: Cada loteria (Mega-Sena, Loto Fácil, Quina, Lotomania) possui seu próprio gerador que permite aos usuários excluir números indesejados e selecionar suas preferências. Algoritmos JavaScript geram combinações aleatórias respeitando as regras específicas de cada jogo, oferecendo controle total sobre a seleção dos números. Desobramentos: Ferramentas matemáticas que calculam combinações otimizadas com base em tabelas de garantia. Os usuários selecionam de 7 a 15 números e escolhem entre uma garantia de quatro números (4 números corretos) ou uma garantia de cinco números (5 números corretos). O sistema gera automaticamente o número correto de apostas necessárias para atingir a garantia escolhida, calculando os custos e organizando as apostas. Últimos Resultados: Microsserviços independentes que consomem a API oficial da Caixa Econômica Federal para recuperar e exibir os dados mais recebidos

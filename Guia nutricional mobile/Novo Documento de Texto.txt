Aplicativo Completo de Saúde e Fitness O Guia Nutricional é um aplicativo React Native desenvolvido com Expo que oferece uma solução completa para gerenciamento de saúde, nutrição e fitness. O projeto utiliza tecnologias modernas como TypeScript, NativeWind (Tailwind CSS), AsyncStorage para persistência de dados e React Navigation para navegação. Arquitetura e Tecnologias O aplicativo é construído com React Native + Expo para desenvolvimento multiplataforma TypeScript para tipagem estática e melhor desenvolvimento AsyncStorage como banco de dados local React Navigation para navegação entre telas NativeWind para estilização com Tailwind CSS Expo SQLite para armazenamento estruturado Sistema de Autenticação O app possui um sistema completo de autenticação Registro com validação de dados pessoais (nome, email, senha, data de nascimento, peso, altura, gênero, nível de atividade e objetivo) Login com username e senha Verificação automática de autenticação ao abrir o app Perfil personalizado com cálculo automático de idade e necessidades calóricas Funcionalidades Principais 1. Planos Alimentares Personalizados Geração automática de planos baseados no objetivo do usuário (perder peso, manter, ganhar peso ou músculo) Cálculo preciso de calorias diárias considerando peso, altura, idade, gênero e nível de atividade Distribuição equilibrada de macronutrientes (proteínas, carboidratos e gorduras) Visualização detalhada de refeições com informações nutricionais 2. Biblioteca de Receitas 500+ receitas low-carb organizadas por categoria 500+ receitas low-calorie divididas por tipo de refeição (café da manhã, almoço, jantar, lanches, sobremesas) Informações nutricionais detalhadas para cada receita Sistema de busca e filtros 3. Planos de Exercícios Geração personalizada baseada no objetivo do usuário Exercícios específicos para cada meta (perda de peso, ganho de massa muscular, etc.) Cronograma semanal de treinos Dicas e orientações personalizadas 4. Acompanhamento de Progresso Registro de peso com histórico completo Visualização de tendências e progresso Estatísticas detalhadas Possibilidade de adicionar notas aos registros Recursos Adicionais Calculadora de IMC integrada Sistema de assinatura com plano gratuito (3 planos alimentares e 3 planos de exercícios) Configurações personalizáveis do usuário Interface moderna com tema vermelho unificado (#dc2626) Navegação intuitiva entre todas as funcionalidades Persistência de Dados O aplicativo utiliza AsyncStorage para armazenar Perfil completo do usuário Histórico de progresso Planos alimentares e de exercícios gerados Configurações de assinatura Dados de autenticação Fluxo do Usuário Primeiro acesso Registro completo com questionário de perfil Login diário Acesso rápido com username e senha Dashboard principal Visão geral de todas as funcionalidades Geração de planos Personalizados baseados no perfil Acompanhamento Registro contínuo do progresso Biblioteca de receitas Acesso a centenas de opções saudáveis O Guia Nutricional oferece uma experiência completa e personalizada para quem busca melhorar sua saúde e fitness, combinando tecnologia moderna com funcionalidades práticas e úteis para o dia a dia.     https://youtu.be/3_uZiSALiMU

MVP de plataforma mobile para monitoramento de passagens aéreas, com autenticação, busca de voos, criação de alertas e histórico. O app permite cadastro/login com JWT, pesquisa por origem/destino com data no padrão brasileiro (DD-MM-AAAA), opção de ida e volta, exibição de resultados em tela dedicada, detalhamento de preço total/ida/volta e acesso a ofertas via deep links. É possível criar alertas de preço-alvo por rota, listar/excluir alertas e disparar notificação local de teste. O histórico de buscas é registrado e consultável pelo usuário.
Arquitetura em monorepo. Backend em FastAPI com SQLite, SQLAlchemy e Alembic, validação com Pydantic, autenticação JWT (python-jose, passlib), geração de dados e previsão de preço com scikit-learn, pandas e joblib. Containerização com Docker e Docker Compose. Frontend em React Native com Expo (SDK 53) e TypeScript, UI com react-native-paper, navegação com react-navigation (stack/tabs), armazenamento seguro de token com expo-secure-store, notificações locais com expo-notifications, chamadas HTTP via axios. Build Android com Dev Client, Gradle e JDK 17. Automatizações em Windows com scripts .bat. O projeto inclui configuração de CORS, middleware, máscara de data, novas telas de resultados, links de oferta e ajustes de layout responsivo.
